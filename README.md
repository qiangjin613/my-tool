# My Tools
开发工作效率的小工具

### mybatis generator
#### 方式 1
idea右边导航栏的“Maven”按钮 --> my-tool --> Plugins --> mybatis-generator --> 单击“mybatis-generator:generate”

#### 方式 2
使用命令，在 idea 底部导航栏 Terminal 中键入命令
```
mvn mybatis-generator:generate
```
